# Evolved Snake

## Vercel Link
https://evolved-snake-game-taupe.vercel.app/

## Design Concept
當老師提出題目時，我立刻聯想到《Undertale》。若將『貪吃蛇』的成長機制與 Boss 彈幕閃避結合，一定很有趣。在我的構想中，玩家需在躲避密集彈幕的同時蒐集道具，隨著蛇身變長，移動空間會愈發狹窄險峻。這徹底顛覆了傳統玩法，將其轉化為具備挑戰性與節奏感的 Boss 戰。這不僅是程式作業，更是個結合邏輯規劃與危機處理的有趣實驗，我對此充滿期待。

## How My Version Is Different
增加了Boss  
增加了三種球 攻擊球 護盾球 回血球  
增加四個狀態顯示 血量 護盾 長度 boss血條  
增加了兩個玩家技能   
增加了暫停功能  
增加了Boss 二階段  
增加了受傷保護  
增加了bgm 聲音反饋  

## How To Play
上下左右or WSAD移動
